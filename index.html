The HTML Cheatsheet

HTML Elements:

Tag Description MDN Link
<(example tag)></(example tag)> Most HTML elements have an opening and closing tag as shown on the left. Note: the following elements do have a closing tag except for any that will be pointed out. < href="https://developer.mozilla.org/en-US/docs/Learn/Getting_started_with_the_web/HTML_basics" target="_blank">HTML Basics
<body> The <body> tag encloses all the HTML code that is rendered on a website. You can only have one per HTML document. The Body Tag
<h1>, <h2>, <h3>, <h4>, <h5>, <h6> In HTML these are different levels of heading ordered from the highest (<h1>) to the lowest (<h6>). Heading Elements
<div> The <div> tag is used as a container for sections of HTML content. Divs can contain anything that can be encased in the <body> element, which is known as flow content. The Content Division Element
id="" The id="" attribute uniquely identifies elements in your HTML code and helps you to style specific elements of it with CSS and JavaScript. ID attributes should only contain letters (a-Z), numbers (0-9), hyphens (-), underscores (_) and periods (.). This tag is placed inside of the opening tag of an HTML element, e.g. <h1 id="title">. The ID Attribute
<span> The <span> tag is used as an inline container for text and can be used to style elements within paragraphs (e.g. links). It is used generically in HTML to separate pieces of text from paragraphs and should only be used if no other semantic element is available. The Span Element
<p> The <p> element is used to display paragraphs in a website as a block of text. The Paragraph Element
<em> The <em> tag emphasises the text between its opening and closing tags and browsers will generally render it as italic. The Emphasis Element
<strong> The <strong> tag highlights important information to a user and browsers will render it as bold by default. The Strong Importance Element
<br> Line breaks can be created with this <br> tag. It must not have a closing tag. The Line Break Element
<li> The <li> element displays elements in a list inside unordered lists (<ul>) and ordered lists (<ol>). The List Item Element
<ul> The <ul> element displays lists in no particular order, as bullet points. The Unordered List Element
<ol> The <ol> element is used to display lists in order with numbers. The Ordered List Element
<img src=""> You can add images with the <img> tag, which is self-closing. It has a source attribute to specify where the image is located, whether that it is a website or a relative file path. It goes after the letters img in the opening tag like so: <img src="https://example.com/file/image.jpg> The Image Embed Element
<img src="" alt=""> Image alts allow you to display alternative text when an image fails to load on a website. The text will also be read aloud if a screen reader is used. The Alt Image Attribute
<video src="" controls> HTML also allows you to add videos to your webpages with the video element (which isn't self-closing). You can add the link to the video inside of the source attribute and the controls attribute adds video controls to it. Any text within the opening and closing tags of the video element will be rendered on the website if the video is unable to load. The Video Embed Element
